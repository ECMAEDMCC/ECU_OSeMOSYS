# Estructura del Repositorio

El presente repositorio tiene tiene la siguiente estructura:

- **readme.md**: Archivo de bienvenida con una descripción del proyecto.

- **Documentacion/**: Carpeta con la documentación del proyecto.

- **AgriculturaUSCUSS/**: Código fuente del modelo de *Agricultura* y *USCUSS*.

- **Energia/**: Código fuente del modelo de *Energía*.

- **ProcesosIndustriales/**: Código fuente del modelo de *Procesos Industriales*.

- **Residuos/**: Código fuente del modelo de *Residuos*.

- **requirements.txt**: Dependencias del proyecto (para poder ejecutar
  la simulación, son librerías de Python).

- **environment.yml**: Entorno anaconda utlizado para ejecutar el proyecto (Recomendado en lugar de requirements.txt).

- **.github/**: Configuraciones para CI/CD, publicación de la documentación.

**Cómo Explorar Archivos en GitHub**

1.  **Usar la Barra de Búsqueda** - En la página del repositorio, usa la
    barra de búsqueda para encontrar archivos o directorios específicos.

2.  **Ver el Historial de Cambios** - Cada archivo tiene un historial de
    cambios accesible desde la pestaña «History» o en la opción *\#
    Commits*, bajo el botón verde \<\> Code.

3.  **Descargar el Código** - Puedes descargar el repositorio en
    formato *.zip* presionando clic izquierdo el botón verde Code y
    después clic izquierdo en la opción Download ZIP. También se puede
    clonar con el siguiente comando por medio de *Git Bash*: 
    ```bash
        git clone https://github.com/PLANACCION-PLANMICC/ECU_OSeMOSYS
    ```
